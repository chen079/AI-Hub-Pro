1. 项目简介

AI Hub Pro 是一个现代化的 AI 模型聚合客户端。它允许你通过一个统一的界面，连接并使用市面上几乎所有的 AI 模型（如 GPT-4, Claude 3.5, DeepSeek, Midjourney 等）。

本项目不提供 AI 模型服务，而是作为“连接器”。你需要自备 API Key（支持 OneAPI/NewAPI 等中转服务）。

2 安装与启动

方式一：Windows 一键启动 (推荐)

直接在项目根目录下双击运行 run.bat 脚本。

  • 它会自动检测 Python 环境。
  • 自动创建虚拟环境并安装依赖。
  • 自动打开浏览器访问系统。

方式二：手动启动 (Mac/Linux)

1. 安装依赖

pip install -r requirements.txt

2. 启动服务

python app.py

3 API 配置

这是使用系统的第一步。登录后，点击左下角的头像，选择 系统设置 (Settings)。

API Endpoint (接口地址)

AI 服务的请求地址。

  • OpenAI 官方: https://api.openai.com/v1
  • 中转服务 (OneAPI): https://你的域名.com/v1

API Key (密钥)

以 sk- 开头的密钥串。

您的密钥在本地数据库中加密存储。

💡 验证技巧

填写完信息后，请务必点击 🔌 测试连接 按钮。系统会尝试连接服务器并获取模型列表，确保配置无误。

4 功能使用指南

流式对话

像使用 ChatGPT 一样。支持 Markdown 渲染、代码高亮。如果模型支持（如 DeepSeek R1），系统会自动折叠显示 <think> 深度思考过程。

多模态展示

图片： 如果使用了 Midjourney/Dall-E 模型，图片会直接显示。
视频： 使用 Luma/Runway 生成的视频会自动加载播放器。
音频： Suno 音乐会自动加载音频条。

文档上传 (RAG)

点击输入框旁的 别针图标，上传 PDF、Word 或 TXT 文件。系统会自动解析文字内容并发给 AI，让 AI 基于文档回答问题。

6 常见问题

Q: 为什么点击“测试连接”提示 404？
通常是因为 Endpoint 地址写错了。请确保地址结尾没有多余的斜杠，且路径正确（例如 /v1）。
Q: 如何在局域网内让手机访问？
修改 app.py 最后一行，将 app.run(port=5000) 改为 app.run(host='0.0.0.0', port=5000)。然后确保电脑防火墙允许 5000 端口，手机访问电脑 IP:5000 即可。
